spinner-verbs

Themed spinner verb sets for Claude Code

| Loading...

Claude Code shows a spinner while it thinks. By default it cycles through generic verbs like "Thinking" and "Analyzing". spinner-verbs lets you replace those with themed sets -- from Dutch TV references to studio chaos.

Browse Sets

Install

install globally
bun install -g spinner-verbs

see available sets
spinner-verbs list

apply a set
spinner-verbs install <name>

Create Your Own

Got a theme in mind? Adding a set takes just three steps.

1
Fork

Fork the repo and clone it locally

2
Create

Add a JSON file in sets/<lang>/ with your verbs

3
PR

Open a pull request and share it with the community
